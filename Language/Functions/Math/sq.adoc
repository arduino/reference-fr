= sq(x)


// OVERVIEW SECTION STARTS
[#overview]
--

[float]
=== Description

[%hardbreaks]


[float]
=== Syntax
`sq(x)`


[float]
=== Parameters
`x`: the number, any data type

[float]
=== Returns
The square of the number. (double)
